================================================================================
                    Spot Zero Ambassador 봇 사용 설명서 (유저 체크용)
================================================================================

■ 봇이 하는 일
  - 퀘스트(미션) 제출 → 관리자 승인 → XP 획득 → 티어/역할 자동 부여
  - 제출한 링크는 관리자 채널로 전달되며, 승인/반려 후 결과를 DM으로 알려줍니다.


■ 슬래시 명령어 (3개)

  1) /sz
     - 에이전트 상태 보드(퀘스트 보드)를 연다.
     - 내 랭크, 총 XP, 다음 티어까지 진행 바, 활성 미션 목록을 볼 수 있다.
     - 아래 드롭다운에서 미션을 선택한 뒤, 링크(증거)를 입력해 제출한다.
     - 제출 후 "24시간 내 확인" 안내가 나오며, 승인/반려는 관리자 채널에서 처리된다.
     - 반려된 제출이 있으면 같은 화면에서 반려 사유를 확인할 수 있다.

  2) /ranking
     - 서버 내 Spot Zero 에이전트 리더보드(상위 10명)를 본다.
     - 1~3위는 강조 표시, 4~10위는 터미널 스타일로 표시된다.
     - 자신이 10위 밖이면 맨 아래에 내 순위와 XP가 따로 표시된다.

  3) /log
     - 내 XP 획득 이력을 최근 15건까지 본다.
     - 각 항목: 날짜/시간, 미션 이름, 획득 XP가 표시된다.
     - 기록이 없으면 "기록 없음" 메시지가 나온다.


■ 퀘스트(미션) 종류

  [직접 제출 퀘스트] — /sz에서 미션 선택 후 링크 제출
  
    • Mission A: SNF Promo Video — 150 XP (1회만)
    • Mission B: Upload 1 Video — 80 XP (반복 가능)
    • Mission C: Live Stream 1 Time — 100 XP (반복 가능)
    • Mission H: High Engagement — 1,500 XP (1회만, 스냅샷 필요)

  [마일스톤 퀘스트] — 제출 없음. Mission B/C 승인 개수에 따라 자동 완료
  
    • Mission D: 비디오 5개 승인 시 — 200 XP (1회)
    • Mission E: 비디오 10개 승인 시 — 500 XP (1회)
    • Mission F: 라이브 스트림 3개 승인 시 — 150 XP (1회)
    • Mission G: 라이브 스트림 6개 승인 시 — 300 XP (1회)


■ 티어(랭크) 시스템

  XP에 따라 자동으로 Discord 역할이 부여된다.

    Lv1  Code SZ        0 XP      (기본)
    Lv2  SZ Streamer    0 XP      (가입 시 부여)
    Lv3  SZ Elite       500 XP
    Lv4  SZ Ambassador  1,000 XP
    Lv5  SZ Partner     2,500 XP


■ 제출 → 승인/반려 흐름

  1) /sz 입력 → 퀘스트 보드에서 드롭다운으로 미션 선택 → 모달에 증거 링크 입력 → 전송
  2) 관리자 채널에 제출 글이 올라가고, 관리자가 [승인] 또는 [반려] 버튼을 누른다.
  3) 승인 시: XP 적립, 마일스톤 조건 충족 시 추가 XP 자동 지급, 티어 업 시 역할 자동 부여. DM으로 "승인됨 + 획득 XP" 안내.
  4) 반려 시: 관리자가 반려 사유를 입력. DM으로 "반려됨 + 사유" 안내. /sz에서 반려 내역 확인 가능. 같은 미션은 다시 제출 가능.


■ 유저 체크 포인트 (테스트 시 확인할 것)

  □ /sz 입력 시 퀘스트 보드(에이전트 상태 보드)가 뜨는가?
  □ 드롭다운에서 미션을 선택하면 "Proof URL / Link" 입력 모달이 뜨는가?
  □ 링크 입력 후 전송하면 "Submission received! Admins will review it soon." 메시지가 나오는가?
  □ /ranking 입력 시 리더보드(1~10위 + 본인 순위)가 표시되는가?
  □ /log 입력 시 XP 이력(또는 "No records found")이 표시되는가?
  □ 관리자 승인 후 DM으로 승인 알림이 오는가?
  □ 관리자 반려 후 /sz에서 반려 사유가 보이는가?


■ 문제 발생 시

  - "응답하지 않음"이 뜨면: 잠시 후 다시 시도하거나, 봇이 온라인인지 확인.
  - /log가 안 되면: 이전에 수정된 버전이 배포되었는지(Railway 재배포) 확인.
  - 역할이 안 올라가면: 서버에 Code SZ, SZ Streamer, SZ Elite, SZ Ambassador, SZ Partner 역할이 있고, 봇 역할이 이 역할들보다 위에 있는지 확인.
